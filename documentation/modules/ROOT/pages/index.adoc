== Gloo Hybrid Cloud Demo

A hybrid cloud deployment, that demonstrates how to use Gloo Mesh or Gloo Edge with Istio based microservices. The demo shows how Gloo helps in:

- Providing gateway to API 
- Do rate limiting
- Do failover
